# 🍔 Food Order App

A responsive Food Ordering Web Application built with **React.js** that allows users to browse restaurants, explore menus, and manage their food cart.

---

## 🚀 Features

- 🏠 Home page with restaurant listings
- 🍽️ Restaurant details page
- 📋 Browse restaurant menus
- 🛒 Shopping cart interface
- ⚡ Fast client-side routing using React Router
- 📱 Responsive design
- 📂 JSON-based mock data

---

## 🛠️ Tech Stack

- React.js
- React Router DOM
- JavaScript (ES6)
- HTML5
- CSS3

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/your-username/food-order-app.git
```

Navigate into the project

```bash
cd food-order-app
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm start
```

The application will run at:

```
http://localhost:3000
```
